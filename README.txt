MINESWEEPER MAP TOOL

-----------
DESCRIPTION

This program helps users create a minsweeper map. The user is prompted to enter data such as the grid size of the minsweeper map, amount of mines to place in the map, and the coordinates to place the mines. The user then has a choice to display the minsweeper map they created, see a list of mine locations or restart and create a new map.
